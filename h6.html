How to enable and disable ATM withdrawals, online payments and other transactions for credit and debit cards of popular banks

TIMESOFINDIA.COM / Jul 5, 2023, 12:03 IST

To make debit and credit cards more secure, the Reserve Bank of India has issued a couple of guidelines and features for banks to curb banking fraud and misuse of cards. One of these features is to allow users to enable or disable card transactions for both

domestic and international payments. The guideline also instructed banks to only keep ATMs and point of sale (PoS) transactions enabled by default and disable all kinds of online transactions for both debit and credit cards that have never been used for digital payments for both domestic and international transactions. If you are also looking to manage your card transactions and want to enable or disable a transaction option, here’s our bank-wise guide for you. State Bank of India (SBI) Credit card 1. Using SBI Card app: Login-> Select ‘Manage Card Usage’ -> Select ‘Manage Card Transactions’ -> Choose transaction type to be modified. 2. SBI Card website: Login -> Select ‘Manage Card Usage’ available under the main menu -> Click on ‘Manage Card Transactions’ -> Choose transaction type to be modified. Debit card 1. Login in https://www.onlinesbi.sbi with User Id and Password →Enter OTP received in registered Mobile Number (RMN) →Select Menu → E-Services→ATM Card Services→ATM Card Limit/Channel/Usage Change → Select Account Number→Select Card Number→Select Services→Change Channel Type → Select ATM Channel /POS Channel/CNP 2. Channel (Online) →Select Enable / Disable Radio Button and Submit.
